HOW TO USE THE AUTO-FISHER
==========================
Created by Bphilips 

1. INSTALL LIBRARIES:
   Open PowerShell and run:
   pip install opencv-python numpy pyautogui pillow

2. GET YOUR BOBBER IMAGE:
   - Start Minecraft Bedrock.
   - Set Video settings to "Windowed Mode" (NOT Fullscreen).
   - Cast your fishing rod.
   - Take a screenshot of the game.
   - Open the screenshot in Paint or Snipping Tool.
   - Crop it so ONLY the red/white part of the bobber is visible.
   - Save it as 'bobber.png' inside this folder.

3. RUN THE SCRIPT:
   - Right-click 'main.py' and select "Run with Python" OR
   - Run 'python main.py' in PowerShell.

4. IN GAME:
   - Cast your rod manually the first time.
   - The script will track the bobber and right-click automatically when it sinks.
